# S1 / S2 / S3 Artefact

Version refondue de l'artefact pedagogique pour un usage Git/GitHub.

## Fichiers

- `index.html` : structure de l'interface
- `styles.css` : design tokens, responsive, theming
- `content.js` : contenu FR/EN et donnees pedagogiques
- `app.js` : rendu dynamique et interactions
- `S1-S2-S3_Artefact.html` : version source initiale, conservee intacte

## Usage

Ouvrir `index.html` dans un navigateur pour une lecture locale.

## Intentions de refonte

- meilleure maintenabilite dans Git
- panneaux detail en tiroir desktop et plein ecran sur mobile
- theming clair / nuit
- suppression de l'appel direct a une API tierce depuis le navigateur
- meilleure base pour un futur deploiement GitHub Pages
