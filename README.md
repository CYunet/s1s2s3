# S1 / S2 / S3 Artefact

A pedagogical artifact presenting PhD research on the perceived value of consulting in the age of AI.

Version refondue de l'artefact pedagogique pour un usage Git/GitHub.

## Fichiers

- `index.html` : point d'entree principal de l'artefact
- `styles.css` : design tokens, mise en page, responsive, theming
- `content.js` : contenu theorique, sections et donnees de timeline
- `app.js` : rendu dynamique, navigation par sections, interactions
- `S1-S2-S3_Artefact.html` : copie synchronisee de l'interface pour usage autonome

## Usage

Ouvrir `index.html` ou `S1-S2-S3_Artefact.html` dans un navigateur pour une lecture locale.
